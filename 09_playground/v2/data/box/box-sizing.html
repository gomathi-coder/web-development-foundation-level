Box Sizing Demo

/* Both boxes have width: 200px, padding: 20px, and border: 4px.
Only the box-sizing property is different! */
/* box-sizing: content-box (default) */
Content-box Total width:
200 + 20*2 + 4*2 = 248px
← Total width includes content + padding + border
/* box-sizing: border-box */
Border-box Total width:
200px (width includes padding & border)
← Total width always stays 200px
