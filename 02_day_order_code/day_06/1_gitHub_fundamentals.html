GitHub Fundamentals - Step-by-Step Guide

1. GitHub Account Creation

  1. Go to https://github.com/
  2. Click on Sign up and fill in your details.
  3. Verify your email and follow the steps to set up your profile.

2. Setup Git Locally

  1. Download Git from https://git-scm.com/
  2. Install Git and open Git Bash (or use Terminal on Mac/Linux).
  3. Run the following commands to set your username and email: 
    git config --global user.name "Your Name"
git config --global user.email "you@example.com"

3. Basic Git Commands

  • git init - Initialize a new Git repository
  • git add . - Add all files to staging
  • git commit -m "Message" - Commit changes
  • git push - Push changes to remote repository
  • git pull - Pull latest changes from remote

4. Create and Push a New Repository

  1. Go to your GitHub profile and click New Repository.
  2. Give it a name (e.g. portfolio-site) and create it.
  3. Open Git Bash in your project folder and run: 
    git init
git remote add origin https://github.com/yourusername/portfolio-site.git
git add .
git commit -m "Initial commit"
git push -u origin master

5. Deploy Using GitHub Pages

  1. Go to your repository on GitHub.
  2. Click Settings > Pages.
  3. Under Source, choose main or master branch, then click Save.
  4. Your site will be published at https://yourusername.github.io/portfolio-site/

6. Share Live URL and Collaborate

  • Copy and share your GitHub Pages URL with others.
  • Collaborate by adding team members to your repository settings under Collaborators.

Optional: Drag-and-Drop Upload

  1. Create a repository on GitHub.
  2. Click on "Add file" > "Upload files".
  3. Drag and drop your HTML/CSS files into the browser window.
  4. Click Commit changes.
  5. Then deploy it using GitHub Pages as described above.
