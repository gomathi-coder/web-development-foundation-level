Git & GitHub Practical Quiz

1. Which Git command is used to create a new repository?

git init
git start
git create
git clone
Example: Initialize a new Git repository
git init

2. What is the purpose of `git clone`?

Create a new repository
Copy an existing repository
Push local changes to the remote
Create a new branch
Example: Clone a repository from GitHub
git clone https://github.com/username/repository.git

3. How do you stage changes for commit in Git?

git add .
git commit .
git stage
git push
Example: Stage all changes for commit
git add .

4. Which Git command is used to commit staged changes?

git push
git commit
git merge
git add
Example: Commit changes to the local repository
git commit -m "Your commit message"

5. How do you push changes to a remote repository in Git?

git pull
git push
git push origin master
git commit
Example: Push changes to GitHub
git push origin master

6. What is a GitHub pull request?

A request to merge changes from one branch to another
A request to clone a repository
A request to delete a repository
A request to commit changes
Example: Creating a pull request on GitHub
Go to GitHub > Pull Requests > New Pull Request

7. What is the purpose of `git fetch`?

Download new data from a remote repository
Push changes to the remote repository
Merge branches
Commit changes locally
Example: Fetch updates from the remote repository
git fetch origin

8. How do you deploy a website using GitHub Pages?

Create a repository and enable GitHub Pages in settings
Use `git deploy` command
Push to a branch called `gh-pages`
None of the above
Example: Deploy your site using GitHub Pages
Go to GitHub repository > Settings > GitHub Pages

9. How do you check the status of your Git repository?

git status

Example: Check repository status
git status

10. How do you view the Git commit history?

git log
