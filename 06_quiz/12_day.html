Bootstrap UI

1. What does a Bootstrap .navbar component typically contain?

The navbar includes navigation links, brand name, toggler, and is responsive:

<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">MySite</a>
  <button class="navbar-toggler" type="button">...</button>
  <div class="collapse navbar-collapse">
    <ul class="navbar-nav">...</ul>
  </div>
</nav>
Only links and buttons
Table data and rows
Brand, toggler, and links
Card content and images

2. What is the purpose of Bootstrap .card component?

The .card is used to display content in a flexible and extensible container.

<div class="card">
  <img src="image.jpg" class="card-img-top">
  <div class="card-body">
    <h5 class="card-title">Card title</h5>
    <p class="card-text">Some quick content.</p>
  </div>
</div>
To draw shapes
To style tables
To structure reusable content blocks
To display charts only

3. What is the purpose of the navbar toggler button?

The navbar toggler collapses or expands the menu on smaller screens.

<button class="navbar-toggler" type="button">
  <span class="navbar-toggler-icon"></span>
</button>
To hide the footer
To expand/collapse menu on mobile
To toggle image size
To animate buttons

4. Which of the following is usually included inside a Bootstrap card?

Cards can include headers, footers, images, text, and links.

<div class="card">
  <div class="card-header">Header</div>
  <div class="card-body">Main content</div>
  <div class="card-footer">Footer</div>
</div>
Only images
Form only
Images, text, buttons
CSS only
