Bootstrap Basics Quiz

1. What is the purpose of the Bootstrap grid system?

The grid system is used to create responsive layouts using rows and columns.

<div class="container">
  <div class="row">
    <div class="col-6">Left</div>
    <div class="col-6">Right</div>
  </div>
</div>
To apply animations
To write JavaScript code
To build responsive layouts
To upload media files

3. What does the class .mt-3 do in Bootstrap?

.mt-3 adds top margin spacing. m stands for margin, t is for top, and 3 is the size.

<div class="mt-3">This div has top margin</div>
Adds margin to the left
Adds margin to the top
Adds margin to the bottom
Adds padding

5. Which class makes a container full width and responsive?

.container-fluid spans the entire width of the viewport.
