CSS Effects Quiz

1. How can you create a smooth transition when changing the background color of an element?

Use `transition: background-color 0.3s;`
Use `background-color: smooth;`
Use `background-color: transition;`
Use `transition: all 0.3s;`
Example: Smooth background color transition
.element { transition: background-color 0.3s; }

2. What property is used to apply a 2D transformation to an element?

transition
animation
scale
transform
Example: Applying a 2D transformation
.element { transform: rotate(45deg); }

3. Which value of the `transform` property will scale an element by 1.5 times its original size?

scale(1.5)
scaleX(1.5)
scaleY(1.5)
scale(150%)
Example: Scaling an element by 1.5 times
.element { transform: scale(1.5); }

4. How do you create a transition effect when hovering over a button that changes the font size?

Use `transition: all 0.3s;`
Use `font-size: transition;`
Use `transition: font-size 0.3s ease;`
Use `transform: font-size 0.3s;`
Example: Hover transition on font size
.button:hover { font-size: 20px; transition: font-size 0.3s ease; }

5. What is the correct syntax to apply a rotation transformation of 90 degrees to an element?

transform: rotate(90deg);
transform: rotate(45deg);
rotate: 90deg;
transform: rotate(90);
Example: Applying a 90-degree rotation
.element { transform: rotate(90deg); }
