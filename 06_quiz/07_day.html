Responsive Design Quiz

1. Which of the following is the correct syntax for a media query to target devices with a width of 600px or less?

@media screen and (max-width: 600px) {}
@media width <= 600px {}
@media (max-width: 600px) {}
@media screen(max-width: 600px) {}
Example: Media query for devices with width 600px or less
@media screen and (max-width: 600px) {}

2. How do you make an image responsive so that it resizes with the screen width?

Use `width: 100%;` in the CSS for the image
Set `height: auto;` for the image
Use `max-width: 100%` for the image
Use `min-width: 100%;` for the image
Example: Making an image responsive
img { max-width: 100%; height: auto; }

3. Which of the following viewport units represents 1% of the width of the viewport?

vw
vh
em
rem
Example: Viewport width unit
div { width: 50vw; }

4. What is the correct CSS to apply a style for mobile devices only?

@media (max-width: 768px) {}
@media (min-width: 768px) {}
@media screen and (max-width: 768px) {}
@media only screen (max-width: 768px) {}
Example: Media query for mobile devices
@media (max-width: 768px) {}

5. How do you ensure that a background image is responsive and doesn't overflow the viewport?

Use `background-size: cover;`
Use `background-size: contain;`
Use `background-repeat: no-repeat;`
Use `background-position: center;`
Example: Making a background image responsive
background-size: cover; background-position: center;
